## Résumé

AuRA Tourisme engage une refonte numérique 2026-2030 avec 3 axes :

1. **Plateforme mutualisée** : CMS central, expérience éditoriale unifiée pour leurs 7 sites web B2B/B2C
2. **Design System public** : charte web unique (couleurs, polices, espacements)
3. **Bibliothèque de composants réutilisables** : briques fonctionnelles (APIDAE, contact, newsletter, cartographie), blocs unitaires conformes RGAA/RGESN

## Points clés pour LIAVO

- **APIDAE mentionné explicitement** comme brique fonctionnelle intégrée → LIAVO utilise déjà APIDAE (IDDJ credentials opérationnels, LMDJ souscription en cours)
- **Mutualisation** = ils cherchent à factoriser plutôt que multiplier les outils → LIAVO est exactement ça pour les séjours
- **Numérique responsable** : certification RGAA AA + RGESN d'ici 2028, adhésion INR → alignement possible avec les valeurs LIAVO (hébergement France, RGPD)
- **Micro-éditeur** : l'ART veut devenir éditeur de ses propres briques → LIAVO pourrait être la brique "séjours scolaires et jeunesse"
- **Écosystème partenaires** : ils mentionnent "le choix de nos prestataires" → opportunité de se positionner comme prestataire technologique

## Opportunités concrètes

1. **Se positionner comme prestataire/partenaire techno** de l'ART sur le volet "séjours jeunesse"
2. **Intégration APIDAE** comme argument de compatibilité avec leur écosystème
3. **Label numérique responsable** (INR, RGESN) → si LIAVO obtient ces labels, ça renforce le positionnement
4. **Financement possible** via les budgets ART dédiés à cette stratégie numérique
